大米发酵就是清酒
糯米发酵就是黄酒
大麦发酵就是啤酒再蒸馏就是威土忌
葡萄发酵就是葡萄酒 加气就是香槟 蒸馏就是白兰地
龙舌兰发酵加蒸馏就是龙舌兰酒
甘蔗发酵再蒸馏就是朗姆酒 能调各种好看还好喝的鸡尾酒
土豆发酵加蒸馏加过滤就是伏特加使劲蒸馏就是96度的生命之水

```mermaid
%% 3. 啤酒 → 威士忌
flowchart TD
    subgraph 啤酒 Beer
        L([大麦麦芽<br>±其他谷物]) --> M[糖化 & 煮沸加酒花]
        M --> N[发酵<br>5% ABV]
        N --> O((啤酒))
    end

    O --> P[蒸馏至 60–70% ABV]
    P --> Q[橡木桶熟成 ≥3年]
    Q --> R((威士忌))
```

```mermaid
%% 4. 葡萄酒 → 香槟 / 白兰地
flowchart TD
    subgraph 静态葡萄酒 Still Wine

    end

    X --> Y[橡木桶熟成 ≥2年]
    Y --> Z((白兰地 Brandy))
```
