# 第4课：人力管理与知识产权入门

阶段一：基础知识第 4 课 / 共 6 课

> **本课目标**
> 学完本课，你能够掌握从招聘到离职的全流程人力管理要点，了解五险一金的组成和成本，同时掌握商标、著作权、专利三类知识产权的申请时机和方法——让你从一开始就为公司搭好"人"和"知识产权"的基础设施。

## 第一部分：人力管理

## 一、招聘面试

### 通知候选人

| 要点 | 做法 | 为什么 |
|---|---|---|
| **谁打电话** | 以HR名义打，不要以老板名义 | 留有余地——如果候选人不行，HR拒绝比老板拒绝体面 |
| **电话后做什么** | 发短信 + 邮件确认 | 邮件里可以附上公司介绍、岗位说明，显得专业 |

### 面试流程

1. **先让候选人放松**——简单寒暄，倒杯水，别上来就"自我介绍一下"

2. **交代面试安排**——告诉候选人面试时间多久、谁面试、什么流程

3. **正式面试**——围绕岗位要求提问，多听少说

4. **结尾留给候选人提问**——这是考察候选人是否真的了解和关心这个岗位的机会

### 面试打分表

不要凭感觉招人。建议做一张打分表，每个面试官分别打分，最后综合对比：

| 评估维度 | 权重 | 面试官A评分(1-5) | 面试官B评分(1-5) | 加权得分 |
|---|---|---|---|---|
| 专业技能 | 30% |  |  |  |
| 沟通表达 | 20% |  |  |  |
| 团队协作 | 20% |  |  |  |
| 学习能力 | 15% |  |  |  |
| 文化匹配 | 15% |  |  |  |
| **综合得分** | **100%** |  |  |  |

### 发Offer

Offer一定要用**邮件**发送。建议在邮件里加上这句话：

> **Offer确认条款（建议加入）**
> "请在某年某月某日前邮件回复'接受聘用'。回复后即视为已签署录用通知函，逾期未回复Offer自动失效。"
> **好处：**把单向邀约变成双向协议，给候选人危机感，减少拿了Offer又不来的情况。

### 拒绝模板

面试后不发通知是不职业的表现。建议用模板委婉拒绝：

**拒绝模板**  
感谢您参加面试，交流非常愉快。您很优秀，只是目前匹配度不是特别高，我会把您加入人才库，未来有需要会主动联系您。

## 二、入职与试用期

### 入职当天要签的文件清单

入职当天必须完成以下文件的签署和留存。这不仅是流程，更是**保护公司**——判断员工有无隐瞒欺诈，同时为可能的劳动仲裁留存证据。

| 类型 | 具体文件 | 目的 |
|---|---|---|
| **签字文件** | 员工登记表、入职承诺书 | 确认员工基本信息真实无误 |
| **留存材料** | 身份证复印件、学历证明、离职证明 | 核实背景，防范风险 |
| **签收手册** | 员工手册、各项规章制度 | 证明员工已知晓公司制度（辞退时必备证据） |

### 试用期规定

| 劳动合同期限 | 试用期上限 |
|---|---|
| 3个月 ~ 1年 | 1个月 |
| 1年 ~ 3年 | 2个月 |
| 3年以上 / 无固定期限 | 6个月 |

| 试用期规则 | 内容 |
|---|---|
| **试用期工资** | 不低于正式工资的80% |
| **试用期解除** | 不能胜任可单方解除，不用赔偿 |
| **管理者建议** | 尽量在试用期内全方位测试员工能力，不然转正后再处理就麻烦得多 |

## 三、辞退与离职

### 辞退类型对比

| 类型 | 是否需要赔偿 | 适用条件 |
|---|---|---|
| **过失性辞退** | **不需要赔偿** | 试用期不合格，或严重违反规章制度 |
| **无过失辞退** | **N+1赔偿** | 没有特别理由，就是不想用了 |

> **过失性辞退的两个必要条件（缺一不可）**
> 1. 公司有**明文公示**的规章制度（这就是为什么入职要签员工手册）
> 2. 有员工违反规章制度的**证据**（打卡记录、工作记录、邮件等）
> 没有公示制度、没有证据，就算员工再过分，你也可能要赔钱。这就是"法律不保护在权利上沉睡的人"。

### 离职管理

| 环节 | 做法 | 为什么 |
|---|---|---|
| **离职证明** | 交接完才出具 | 离职证明是员工找下一家公司的必备文件，用它做"关卡"确保交接完成 |
| **交接单** | 逐项签字确认 | 账号密码、门禁卡、钥匙、财务报销、公司资产等，不留尾巴 |

## 四、竞业禁止

| 项目 | 内容 |
|---|---|
| **最长限制期** | 2年 |
| **生效条件** | 不是签了就生效！离职后需每月支付补偿金（≥月工资30%） |
| **成本控制** | 先付几个月，确定员工去向无威胁后停止 |
| **员工违约** | 可起诉，但需先搜集入职竞品公司的证据 |

> **实操建议**
> 竞业禁止主要针对核心高管和技术骨干，普通员工一般不需要签。签了不付钱等于白签，所以要想清楚再签。

## 五、劳动合同

### 不签劳动合同的后果

| 时间 | 后果 |
|---|---|
| 超过1个月不满1年 | 每月支付2倍工资 |
| 超过1年 | 视为已签无固定期限劳动合同 + 每月2倍工资 |

> **铁律**
> 必须在员工**入职1个月内**签订劳动合同。这是劳动法最基础的规则，一旦违反，员工去仲裁基本一告一个准。

### 案例对比

|  | 失败案例（建筑公司） | 成功案例（腾讯） |
|---|---|---|
| **案情** | 辞退员工，被员工告上法庭 | 辞退7年高级工程师，员工不服 |
| **结果** | 公司败诉，赔钱 | 腾讯胜诉，不用赔偿 |
| **原因** | 没有明文公示的上下班时间、没有打卡制度、没有请假制度、没有证据 | 提供了劳动合同、假期管理制度、工资安排、邮件记录、办公楼视频等完整证据链 |

> **启示**
> 从一开始就把规章制度公示出来，做好日常考勤记录。这不需要多大成本，但到了劳动仲裁时就是决定性的。向大厂学习——不是学它们的福利，而是学它们的流程意识。

## 六、五险一金

### 组成

| 险种 | 用途 |
|---|---|
| **医疗保险** | 看病报销，连续缴费越长报销比例越高 |
| **生育保险** | 生育费用报销 + 生育津贴（如北京约3.8万） |
| **养老保险** | 退休后领养老金，缴满15年可领 |
| **工伤保险** | 工伤医疗和经济补偿 |
| **失业保险** | 非自愿离职可领补偿金 |
| **住房公积金** | 低利率买房贷款 |

### 缴费比例

| 项目 | 企业缴纳 | 个人缴纳 |
|---|---|---|
| 养老保险 | 约20% | 8% |
| 医疗保险 | 约8% | 2% |
| 其他社保 | 约2-4% | 约1% |
| 住房公积金 | 5%-12% | 5%-12% |
| **合计** | **约30-40%** | **约10%** |

> **成本实例**
> 税前工资1万 → 员工到手约7千，企业实际支出约1.3万。
> 这意味着招聘一个人，企业实际承担的成本比员工的"期望工资"高出约30%。招人时一定要把这个算进去。

## 第二部分：知识产权

## 七、商标

### 为什么一定要注册商标

| 原因 | 说明 |
|---|---|
| **保护品牌** | 你不注册别人也可以用，甚至别人注册了你就不能用了 |
| **平台入驻** | 微博、微信、抖音、京东、天猫等平台认证都需要商标 |
| **防御投机客** | 有人专门盯着新闻报道抢注品牌名，等你火了再来要你高价买回 |

> **经典案例**
> 滴滴最早用"口字旁的嘀"，被抢注后花了巨大代价切换到"三点水的滴"。如果你不想重蹈覆辙，公司名定下来后第一件事就是查商标、注册商标。

### 商标类型

| 类型 | 说明 | 建议 |
|---|---|---|
| **文字商标** | 产品名、网站名、APP名等——最常见 | 建议文字和图形分开注册 |
| **图形商标** | 经过设计的LOGO | 建议文字和图形分开注册 |
| **声音商标** | 比较少见的类型 | 一般创业公司暂时不需要 |

### 商标注册流程

1. **确定类目**（共45大类，建议1-2个核心类 + 1-2个防御类）

2. **商标查询**（商标局官网 + 第三方平台查重）

3. **准备材料**（营业执照副本、商标标识）

4. **提交申请**

5. **形式审查**（约1个月）→**实质审查**（3-4个月）→**初审公告**（3个月）

6. **拿到商标注册证**

**总周期：**半年到一年。所以商标越早申请越好，不要等产品上线了才想起注册商标。

### 费用对比

| 方式 | 费用/类 | 适合人群 |
|---|---|---|
| 自己在商标局官网提交 | 270元（官费） | 有经验、时间充裕的创业者 |
| 第三方平台（阿里云/腾讯云） | 约320元（官费+50元手续费） | 大部分创业者——性价比高 |
| 找专业顾问 | 270元 + 300-500元服务费 | 对商标不熟悉、想提高通过率 |

### 注册 vs 买商标

| 方式 | 优势 | 劣势 |
|---|---|---|
| **注册** | 成本低（几百元） | 周期长（半年+），可能被驳回 |
| **直接买** | 快速拿到证（1-2周），消除不确定性 | 成本高（几千到几十万不等） |

> **案例**
> 锤子科技为了赶新品发布会，直接买了"坚果"商标，只用了一个月就拿到了商标。如果你急着用商标，买一个现成的也是可行的选择。

## 八、著作权（版权）

### 两大类型

| 类型 | 适用范围 | 典型例子 |
|---|---|---|
| **软件著作权** | 软件类产品 | 网站、APP、后台程序、小程序等 |
| **作品著作权** | 创意类作品 | 美术、文字、摄影、词曲、视频等 |

### 申请著作权的两大好处

| 好处 | 说明 |
|---|---|
| **保护权益** | 登记后维权容易（否则要自己找大量证据证明"这是我先创作的"） |
| **获得扶持** | **6个软件著作权可申请高新技术企业**——享受15%企业所得税 + 研发费用加计扣除 |

### 申请流程

1. 登录版权中心网站注册

2. 填报申请表、准备材料

3. 提交后约1-1.5个月领到证书

> **实操建议**
> 软件著作权建议多平台一起保护（苹果 + 安卓 + 小程序 + 网站），不需要等产品上线，**拿着源代码就能申请**。早申请早拿证，凑6个就可以申请高新技术企业了。

## 九、专利

### 三种专利类型对比

| 类型 | 含金量 | 说明 | 案例 | 保护期限 |
|---|---|---|---|---|
| **发明专利** | 最高 | 全新的技术方案 | 发明一台榨汁机 | 20年 |
| **实用新型** | 中等 | 产品形状/构造的改进改良 | 榨汁机从2小时/个改良到10分钟/个 | 10年 |
| **外观设计** | 较低 | 产品的形状、图案、色彩设计 | 榨汁机是圆的还是方的、印什么图案 | 10年 |

### 申请专利的5大好处

1. **市场保护**——可要求电商平台下线侵权产品

2. **获得收益**——授权他人使用或转让变现

3. **税收优惠**——助力申请高新技术企业（15%企业所得税）

4. **彰显实力**——宣传专利数量提升品牌形象

5. **防御**——防止他人抢先申请反过来告你侵权

### 注意事项

| 注意点 | 说明 |
|---|---|
| **尽早申请** | 产品确定后越早申请越好，不用等产品发布，理论上可行即可申请 |
| **每年续费** | 专利需要每年续费维持有效，忘了续费专利就失效了 |
| **保护期限** | 发明20年，实用新型/外观设计10年 |

> **知识产权布局策略**
> 创业公司不一定所有专利都要申请——成本不低。建议优先注册商标（几百元必备），有技术创新的申请实用新型专利，有软件产品的申请软件著作权(凑6个拿高新资质)。发明专利费用高、周期长，有核心技术的再考虑。

## 课后思考

### 思考题

1. 你公司的员工手册和规章制度有没有公示并让员工签收确认？如果现在有人来仲裁，你能拿出证据吗？

2. 算一笔账：如果你打算招一个期望月薪1万的员工，公司实际每月要承担多少成本？

3. 你的公司名称有没有去查商标？有没有被抢注的风险？

4. 你现在的产品/服务能申请什么类型的知识产权？列出清单。
