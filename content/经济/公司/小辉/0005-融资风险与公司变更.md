# 第五课：融资、风险与公司变更

阶段一：基础知识第 5 课 / 共 6 课

> **本课目标**
> 学完本课，你能够判断自己是否需要融资、了解投资人类型和融资全流程、掌握BP写作的核心方法，同时熟悉广告法合规要点和公司变更、注销的基本流程——让你的创业之路在"拿钱"和"合规"两个关键环节上不出错。

## 第一部分：融资要领

## 一、你真的需要融资吗

很多创业者把"拿到融资"当作成功的标志。但在决定融资之前，先问自己几个问题：

| 问题 | 思考方向 |
|---|---|
| 现在的问题是否必须用钱解决？ | 很多问题（如产品方向、市场策略）不是钱能解决的 |
| 需要多少钱？能真正解决问题吗？ | 融资要有明确用途，而不是"先拿了再说" |
| 是否愿意承担拿钱后的增长压力？ | 拿了投资人的钱，就要对增长预期负责 |
| 愿意出让多少股份？ | 出让股份 = 损失部分分红权和控制权 |

> **融资双刃剑**
> **好处：**有钱加速发展、获得资源和人脉、品牌背书
> **代价：**增长预期压力、股份稀释、控制权可能旁落、失去部分决策自由
> 拿钱不是免费的——只是代价不在今天支付。

## 二、投资人类型

不同阶段的创业者适合找不同类型的投资人。选对投资人比拿到钱更重要。

### 按性质分类

| 投资人类型 | 特点 | 适合阶段 |
|---|---|---|
| **个人投资人（天使投资人）** | 个人出资，决策快，投后管理少 | 种子轮、天使轮 |
| **VC（风险投资）** | 机构化运作，有专业团队，但要求高增长 | 天使轮到C轮 |
| **战略投资人** | 大公司旗下的投资部门，看重业务协同 | B轮以后 |
| **政府引导基金** | 政府出资，扶持特定行业，要求不高但流程长 | 成长期以后 |
| **孵化器/加速器** | 提供种子资金+办公空间+资源对接 | 种子轮、天使轮 |

### 按币种分类

| 类型 | 特点 | 典型机构 |
|---|---|---|
| **美元基金** | 投资规模大，接受VIE架构，追求高回报，投后要求严格 | 红杉资本、IDG资本、启明创投 |
| **人民币基金** | 投资相对保守，要求内资架构，退出方式多样 | 深创投、达晨创投、各地方政府基金 |

### 第一笔融资从哪来

> **3F原则**
> 很多创业者的第一笔钱来自3F：Family（家人）、Friends（朋友）、Fools（"傻瓜"——指信任你的早期支持者）。这是最现实也最可行的融资起点。

## 三、投资阶段：从种子轮到IPO

融资有清晰的阶段划分，每个阶段的投资金额和关注点不同：

| 阶段 | 公司状态 | 典型融资金额 | 投资人关注 |
|---|---|---|---|
| **种子轮** | 只有想法/原型 | 几十万-几百万 | 团队背景、市场潜力 |
| **天使轮** | 产品初步上线 | 几百万-千万 | 产品方向、用户反馈 |
| **A轮** | 产品验证通过 | 千万-亿级 | 商业模式、增长数据 |
| **B轮** | 快速扩张中 | 亿级 | 市场份额、运营效率 |
| **C轮及以后** | 行业领先 | 数亿-数十亿 | 盈利能力、上市可能 |
| **IPO（上市）** | 公开募股 | 视市值而定 | 公开市场投资者 |

## 四、商业计划书（BP）写作

### BP的核心目标

很多创业者花几个月写一本厚厚的商业计划书，但投资人的真实场景是——**在手机上扫一眼，不超过3分钟**。

> **BP的目标不是讲清所有业务逻辑，而是引起投资人的兴趣。**
> 让投资人想约你见面聊，BP就完成了使命。

### 引起投资人兴趣的4个要素

| 要素 | 说明 |
|---|---|
| **定位清晰垂直** | 别定太大，投资人在小赛道里看到大机会才感兴趣 |
| **赛道好** | 趋势好、空间大、竞争少、增长快——满足2-3个即可 |
| **账能算明白** | 有清晰的财务模型——收入怎么来、成本怎么花、什么时候盈亏平衡 |
| **团队亮眼** | 投资本质是投人——背景、经验、执行力 |

### BP的9页标准结构

| 页码 | 内容 | 一句话说明 |
|---|---|---|
| 第1页 | **一句话定位** | 用10秒钟让投资人知道你是做什么的 |
| 第2页 | **用户痛点** | 场景化描述——让投资人感同身受 |
| 第3页 | **解决方案** | 你的产品/服务怎么解决这个痛点 |
| 第4页 | **业务数据** | 用户数、增长率、转化率等关键指标 |
| 第5页 | **商业模式** | 怎么赚钱——收入来源、成本结构 |
| 第6页 | **市场规模** | 天花板在哪——用权威数据支撑 |
| 第7页 | **竞品对比 + 竞争优势** | 你和别人有什么不一样、为什么你能赢 |
| 第8页 | **团队介绍** | 核心成员的背景和分工 |
| 第9页 | **融资目标和用途 + 联系方式** | 要多少钱、花在哪、怎么联系你 |

### 一句话定位的两种写法

#### 对标法

"篮球版的抖音"——投资人立刻有具象想象

公式：**XX版的YY**

- 让投资人3秒理解

- 选对标的很重要

- 别对标太大的（"下一个阿里巴巴"）

#### 描述法

"为初创企业提供一站式企业服务的平台"

公式：**客群 + 模式 + 属性**

- 更准确严谨

- 适合模式较新的项目

- 但不够生动

## 五、融资阶段流程

融资是一个标准化的流程，一般经历以下5个阶段：

| 阶段 | 内容 | 建议 |
|---|---|---|
| **1. 前期准备** | 行业研究、业务梳理、写BP | 做好功课再出发，准备好数据 |
| **2. 路演跟进** | 推荐投资人→路演→签保密协议→初步尽调 | 不要只找一家，多轮并行推进 |
| **3. Term Sheet** | 谈核心条款（投资额、估值、控制权等）→签订 | 估值不是最重要的，条款更重要 |
| **4. 尽职调查** | 财务/法务/业务进场调查 | 提前整理好材料，别等调查时才找 |
| **5. 正式签约** | 律师把关→签订文件→股份和资金交割 | 一定要找有经验的律师审查合同 |

> **关键提醒**
> Term Sheet阶段最容易出问题。估值、董事会席位、一票否决权、清算优先权、对赌条款——每个条款都可能有隐藏的坑。建议找有融资经验的律师帮你审一遍。

## 第二部分：风险意识与公司变更

## 六、风险意识

### 广告法合规

很多创业者在宣传时为了吸引眼球，会不自觉地使用一些"极限用语"。这在广告法中是明确禁止的。

| 禁用类型 | 例子 | 说明 |
|---|---|---|
| **极限用语** | "最"、"第一"、"国家级"、"最佳"、"顶级" | 不能证明的绝对化描述都算违规 |
| **虚假宣传** | 宣传专利数量但专利还没拿到证书 | 申请中的专利不能算数 |

> **真实案例**
> 有创业公司在官网写了"全网最好吃的坚果"、"销量第一"等宣传语，被竞争对手举报到工商局，被处以数万元罚款。这样的罚款对初创公司是完全不必要的损失。

**对外宣传的"三不"原则：**

- **不用**"最"字头——最好、最大、最便宜

- **不用**"第一"——销量第一、口碑第一

- **不用**无法证明的数据——没有来源的百分比、排名

### 留存证据的习惯

跟人合作（不管是和合伙人、供应商还是客户），都要养成**留存证据**的习惯：

| 场景 | 建议做法 |
|---|---|
| 口头沟通 | 事后邮件或聊天记录确认："刚才我们沟通的内容是……" |
| 合同签署 | 纸质版签字盖章，电子版存云盘 |
| 付款转账 | 备注清楚用途，留好转账记录 |
| 日常管理 | 考勤记录、审批单、邮件往来都存档 |

> **一句话总结**
> 所有可能产生争议的事情，都要有**文字记录**。微信聊天记录也是证据，但不如邮件正式。重要事项尽量用邮件确认。

## 七、公司变更

公司注册好以后，不是一成不变的。以下变更类型是创业过程中最常见的：

| 变更类型 | 场景 | 难度 |
|---|---|---|
| **地址变更** | 搬了实际办公地址后要及时变更，否则工商年检可能异常 | 简单 |
| **法人变更** | 法人更换（比如创始人退出，换人接手） | 一般 |
| **股权变更** | 股东转让股份给其他人 | 一般（可能涉及税务问题） |
| **注册资本变更** | 增资或减资 | 增资简单（基本不交税），减资较复杂 |
| **经营范围变更** | 增加新的业务项目 | 简单 |

> **提醒**
> 公司变更一定要及时办理工商登记。比如你搬了地址但没变更，工商部门联系不到你，公司可能被列入"经营异常名录"，影响信誉和业务开展。

## 八、公司注销

如果公司不做了，**一定不能直接不管**。必须及时注销或转让，否则有严重的法律风险。

### 注销的前提

| 情况 | 能否注销 | 说明 |
|---|---|---|
| **无外债** | 可以正常注销 | 无需补齐注册资本 |
| **有外债** | 需先还完外债才能注销 | 注册资本认缴的部分可能需要补齐用于还债 |

### 注销流程

1. **清算**——成立清算组，清理资产和债务

2. **税务注销**——到税务局办理，结清税款

3. **工商注销**——到市场监督管理局办理

4. **银行注销**——注销公司银行账户

5. **社保注销**——注销社保账户

> **严重警告**
> 公司不做了不注销也不转让，时间长了会被列入**严重违法失信企业名单**，法定代表人会受到限制——不能坐高铁飞机、不能贷款、不能担任其他公司的法人。这不是吓唬人，这是真实的法律后果。

## 课后思考

### 思考题

1. 你的创业项目目前需要融资吗？如果需要，估一下大概需要多少钱、愿意出让多少股份？

2. 尝试用一句话（不超过20个字）说清楚你的项目定位——试试对标法或描述法。

3. 翻一下你的官网、公众号、宣传材料，有没有可能违反广告法的用语？

4. 如果你现在有一家不经营的公司，你知道怎么处理它吗？注销还是转让？
