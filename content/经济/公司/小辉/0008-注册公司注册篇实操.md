# 第8课：注册公司 · 注册篇实操

阶段二：注册实操第 8 课 / 共 11 课

> **本课目标**
> 学完本课，你将清楚自己该选什么企业类型、怎么起名、注册资本和股份怎么定、地址和经营范围怎么填——能够独立完成营业执照的申请和印章办理。

## 一、去哪里办理营业执照

### 线上办理（推荐，全国80%以上城市支持）

1. 搜索"城市名 + 市场监督管理局"（原工商局）

2. 认准带"官方"标识的网站

3. 找到"企业登记"→ "企业设立登记"→ "网上办理"

4. 登录个人中心（未注册公司前用个人账号登录）

如果市级网站找不到入口，搜索"省份名 + 市场监督管理局"。部分城市需要IE浏览器或360极速浏览器兼容模式。

### 线下办理

- 地图搜索"区名 + 市场监督管理局"

- 至少带身份证，可能需要跑2~3次

- 第一次领材料，填完后提交

## 二、选择企业类型

### 主流企业类型对比

| 对比项 | 有限责任公司 | 个人独资企业 | 个体工商户 |
|---|---|---|---|
| 股东人数 | 1~50人（建议2人以上） | 1人 | 1人 |
| 承担责任 | 有限责任 | 无限连带 | 无限连带 |
| 主要税种 | 增值税+企业所得税+个税 | 增值税+个税 | 增值税+个税 |
| 入驻电商 | 天猫/京东/拼多多均可 | 均可 | 很多限制（不能开天猫、京东自营等） |
| 员工数量 | 无限制 | 无限制 | 最多8人 |
| 能否转让 | 可以 | 可以 | 不可以 |
| 分支机构 | 可设分公司/子公司 | 不可以 | 不可以 |

### 其他类型说明

| 类型 | 说明 |
|---|---|
| 股份有限公司 | 股东2人以上无上限，股份可自由转让，适合上市/准上市公司 |
| 合伙企业 | 会计师事务所、律师事务所等特殊行业使用；也用于搭建持股平台 |
| 一人有限公司 | 有可能承担无限连带责任（需证明个人资产与公司资产独立） |

### 税负对比案例

**情况一：年营收100万，利润20万**

| 类型 | 增值税 | 企业所得税 | 个人所得税 | 总税额 |
|---|---|---|---|---|
| 有限责任公司 | 0（免征区间） | 1万（5%） | 3.8万（20%分红税） | 4.8万 |
| 个人独资/个体户 | 0 | 0 | 2.95万（5%~35%累进） | 2.95万 |

**情况二：年营收1000万，利润200万**

| 类型 | 增值税 | 企业所得税 | 个人所得税 | 总税额 |
|---|---|---|---|---|
| 有限责任公司 | 同 | 12.54万 | 32.57万（20%分红税） | ~45万 |
| 个人独资/个体户 | 同 | 0 | 74.6万+（35%累进） | ~75万 |

> **结论**
> 收入越高，有限责任公司反而更划算（查账征收下）。起步阶段个体户/个人独资税负低，但未来要发展就选有限责任公司。

## 三、公司起名

### 公司名的四要素

以"北京赚它一个亿科技有限公司"为例：

- **地区：**北京（也可放字号后面加括弧）

- **字号：**赚它一个亿（核心辨识度）

- **行业：**科技（根据主营业务选择）

- **组织形式：**有限公司

### 三种形式

| 形式 | 示例 |
|---|---|
| 地区 + 字号 + 行业 + 组织形式 | 北京赚它一个亿科技有限公司 |
| 字号 +（地区）+ 行业 + 组织形式 | 赚它一个亿（北京）科技有限公司 |
| 字号 + 行业 +（地区）+ 组织形式 | 赚它一个亿科技（北京）有限公司 |

### 重要原则：公司名 ≠ 商标名

| 对比项 | 公司名 | 商标名 |
|---|---|---|
| 保护范围 | 同地区同行业不可重名 | 全国唯一 |
| 管辖部门 | 市场监督管理局 | 知识产权局 |
| 建议 | 起好公司名后，一定要同时注册商标 |

### 起名思路参考

| 思路 | 案例 | 优势 | 劣势 |
|---|---|---|---|
| 古诗词/俗语 | 百度（众里寻他千百度）、迅雷 | 优雅有内涵 | 好词已被注册 |
| 品类词+特征词 | 美团、大众点评 | 直接占据品类 | 未来不好扩展 |
| 场景特征 | 滴滴、饿了么 | 好注册商标 | 同样不好扩展 |
| 创始人名字 | 小鹏、理想、奔驰 | 创始人+品牌联动 | 创始人变动时尴尬 |
| IP名字 | 阿里巴巴、特斯拉 | 好记好传播 | 知名IP已被注册 |

### 域名建议

- 优先注册品牌名全拼域名（如 alibaba.com）

- 后缀重要性：`.com`>`.cn`>`.net`

- 用品牌名做企业邮箱后缀，显专业

- 可在腾讯云/阿里云直接注册，几十元/年

## 四、注册资本

### 核心认知

**有限责任 = 股东对公司的债务只承担有限责任，最高额度就是注册资本。**

### 三个关键点

| 要点 | 说明 |
|---|---|
| 不需要一次性缴清 | 实行认缴制，承诺时限内缴完即可（可写20~30年），到期可续 |
| 参考行业资质要求 | ICP经营许可证要求100万+，天猫大部分类目要求100万+ |
| 不是越大越好 | 注册资本1000万，欠债1000万需全额承担；注册资本100万，只需承担100万 |

> **建议**
> 先写小一点，未来需要可以增资（增资简单且不交税）。

### FAQ

| 问题 | 回答 |
|---|---|
| 注册资本能花吗？ | 能，但要用公司名义花，有对应票据 |
| 怎么缴？ | 股东个人卡转公司对公账户，备注"入资款"或"投资款" |
| 注销时要补齐吗？ | 无外债不用补，有外债需先还完才能注销 |
| 什么是验资报告？ | 2013年后用得少，招投标时可能需要，由审计师事务所出具 |

## 五、股份比例

### 股权的两大权利

| 权利类型 | 具体权利 |
|---|---|
| **经济权** | 分红权：按股份比例获得公司利润分配 |
| 转让权：将股份转让给他人变现 |
| 优先认购权：其他股东转让时优先购买 |
| 剩余资产分配权：清算时按比例分配资产 |
| **政治权** | 表决权：按股份比例投票决策（最重要） |
| 知情权：了解公司经营和财务信息 |
| 选举管理权：选举董事会成员 |
| 建议质询权：对公司经营提出建议 |

### 控制权的四个关键数字

| 比例 | 含义 | 说明 |
|---|---|---|
| **67%** | 绝对控制权 | 可通过所有重大决策（修改章程、增减资本、合并分立等） |
| **51%** | 相对控制权 | 可通过日常决议 |
| **34%** | 一票否决权 | 可否决重大决策（超过1/3就不能达到2/3） |
| **10%** | 掀桌子的权利 | 可请求质询、调查、起诉、解散公司 |

### 股权分配原则

**原则1：团队要有明确的老大，切忌平均分配**

| 错误分配 | 问题 |
|---|---|
| 50% + 50% | 意见不一致时无人拍板 |
| 51% + 49% | 差距不够大 |
| 33% + 33% + 33% | 三方僵局 |
| 25% × 4 | 四方僵局 |

**原则2：**股东人数2~5人为宜

**原则3：**按综合贡献分配（资金、技术、人力各有权重）

### 一人公司的风险

> **重要风险**
> 一人有限公司（只有一个股东），如果不能证明个人资产与公司资产独立，有可能承担无限连带责任。建议至少找一个人做股东（哪怕只占1%），避免一人公司。

## 六、注册地址

### 基本要求

- 大部分地区要求**商用产权**（写字楼、商铺）

- 住宅一般不行（少数城市如深圳允许但手续复杂）

- 需准备：产权证明（房产证）、租赁合同、租房发票

### 没有实际地址怎么办？

| 方案 | 说明 | 成本 |
|---|---|---|
| 挂靠朋友地址 | 用朋友的商用房子注册 | 免费 |
| 创业园区/孵化器 | 部分园区提供免费注册地址 | 通常免费或低费用 |
| 付费代办地址 | 代办公司提供挂靠地址 | 几百到几千/年 |

> **注意**
> 挂靠地址（异地经营）有风险，一旦自己租了实际场地要**及时做工商变更迁走**。

## 七、经营范围

### 最快方法：找对标公司

在天眼查/企查查搜索同行公司，直接复制其经营范围。**建议找本地公司复制**，因为各地描述可能略有差异。

### 注意事项

- 看不懂的条目可以都放进去，系统会自动过滤不能通过的项

- 部分经营范围需要额外审批（会标注"须经相关部门批准"）

- **可以跨行业经营**，只要经营范围里加上对应项即可

## 八、法人、董事、监事、经理该填谁

| 职位 | 职责 | 谁担任 |
|---|---|---|
| **法定代表人（法人）** | 公司意志的体现人，对公司的行为结果负责，很多工商手续需法人签字 | 建议创始人/实际控制人担任 |
| **执行董事** | 不设董事会时的公司负责人，权力较大 | 通常由法人兼任 |
| **监事** | 监督董事会和经理，防止滥用职权 | 不能与董事/经理同一人，找信得过的人 |
| **经理** | 日常经营管理 | 创始人或外聘 |

> **注意**
> 注册公司至少需要**2人的身份证**（因为监事不能与董事/经理同一人）。初创期监事多为形式上的监督，找信得过的人担任即可。

## 九、公司印章办理

### 办理方式

- **线上申请：**部分地区直接在办理营业执照时选择，办好后寄送

- **线下办理：**拿营业执照去公安局指定的刻章点办理

### 五枚章的用途

| 印章 | 用途 | 保管建议 |
|---|---|---|
| **公章** | 权力最大，工商变更/签合同/证明材料等 | 妥善保管，建议创始人或行政 |
| **合同专用章** | 签合同专用 | 业务部门 |
| **财务专用章** | 财务核算材料 + 银行业务 | 财务/会计 |
| **发票专用章** | 开具纸质发票时使用（电子票不需要） | 财务 |
| **法人名章** | 银行开户、代替法人签字时使用 | 法人或财务 |

## 课后思考

### 思考题

1. 你打算注册什么类型的企业？为什么？（结合你的业务类型和税负考虑）

2. 注册资本你准备写多少？如果写100万，你的责任上限是多少？

3. 如果只有你一个人创业，你会怎么安排股权？怎么避免"一人公司"的风险？

4. 公司名四要素分别是哪四个？你觉得哪一个最重要？

5. 五枚印章中，你最需要保管好哪一枚？为什么？
