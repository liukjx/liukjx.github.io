# 第十课：注册公司 · 社保与综合篇

阶段二：注册实操第 10 课 / 共 11 课

> **本课目标**
> 学完本课，你将掌握五险一金的用途和缴费比例，知道如何为员工办理社保和公积金开户，能够计算税前工资和税后工资，并补齐商标基础认知。

## 一、五险一金用途详解

### 五险

| 险种 | 核心用途 | 要点 |
|---|---|---|
| **医疗保险** | 看病报销（挂号、开药、住院） | 连续缴费越长报销比例越高，**不要断缴**（断缴后连续时间清零） |
| **生育保险** | 生育费用报销 + 生育津贴 | 公立医院基本全报销，津贴按单位人均工资×产假天数算（如北京约3.8万） |
| **养老保险** | 退休后领取养老金 | 缴满15年可领退休金，随平均工资上涨抵抗通胀 |
| **工伤保险** | 工伤医疗和经济补偿 | 认定情形：工作时间/场所受伤、上下班交通事故、职业病等 |
| **失业保险** | 失业后领取补偿金 | 需满足：缴满1年 + 非自愿离职 + 办理失业登记，按最低工资70%~90%发放 |

### 住房公积金

- **用途：**低利率贷款买房（公积金利率约3.25%，商贷约5%）

- **优势：**贷100万分20年还，公积金比商贷少付十几万利息

### 不缴五险一金的法律风险

> **法律依据**
> 社会保险法第84条规定：不办理社保登记的，罚款社保费1~3倍；不按时足额缴纳的，按日加收万分之五滞纳金。

## 二、五险一金缴费比例

| 险种 | 企业缴纳 | 个人缴纳 |
|---|---|---|
| 养老保险 | 约20% | 8% |
| 医疗保险 | 约6%~8% | 2% |
| 失业保险 | 约0.5%~2% | 0%~1% |
| 工伤保险 | 约0.1%~3% | 0 |
| 生育保险 | 约1%（已并入医保） | 0 |
| **社保小计** | **约28%~32%** | **约10%~11%** |
| 住房公积金 | 5%~12% | 5%~12% |
| **合计** | **约30%~40%** | **约10%** |

> **实际影响**
> 税前工资1万 → 员工到手约7千（扣社保+公积金+个税），企业实际支出约1.3万。

## 三、给员工上社保

### 开户流程

1. **开户：**拿营业执照去社保局开设社保账户（一般需线下办理）

2. **首次参保员工：**带身份证去社保局办理，领取社保卡

3. **非首次参保员工：**先让上一家公司做**减员**，你再做**增员**

4. **时间节点：**每月25号前完成增减员（25号后开始扣费不能操作）

### 增员减员操作

| 操作 | 说明 | 操作时间 |
|---|---|---|
| **增员** | 新员工入职时添加社保 | 每月25号前 |
| **减员** | 员工离职时停缴社保 | 每月25号前，需在劳动关系终止后及时操作 |

## 四、给员工上公积金

1. 搜索"城市名 + 住房公积金管理中心"

2. 单位网上业务平台办理

3. 需先去公积金中心开户

4. 选择缴存比例（5%~12%之间，全公司统一）

5. **注意：**一旦设置，每年只有7月可调整一次

## 五、税前工资和税后工资计算

以2020年全国平均月工资8,115元为例：

| 项目 | 金额 |
|---|---|
| 税前工资 | 8,115元 |
| 扣社保（个人约10%） | -892.65元 |
| 扣公积金（5%） | -405.75元 |
| 扣个税（3%） | -54元 |
| **员工税后到手** | **6,762元** |
| 企业社保缴纳（约32%） | +2,596元 |
| 企业公积金缴纳（5%） | +405.75元 |
| **企业实际支出** | **约11,117元** |

> **启示**
> 跟员工谈工资时要让员工知道税后到手多少，自己算成本时不能只算税前工资。

### 缴存比例选择建议

| 比例 | 对企业 | 对员工 | 适合场景 |
|---|---|---|---|
| **5%（最低）** | 成本最低 | 到手工资更高 | 初创期现金流紧张 |
| **12%（最高）** | 成本最高 | 公积金更多，买房利好 | 稳定盈利、员工有购房需求 |

## 六、商标基础快速补齐（与课程③的衔接）

> **快速回顾**
> 以下是对课程③（商标注册）的快速回顾，如果你还没学第6、7课，这部分帮你快速补齐核心知识。

### 商标核心要点速览

| 知识点 | 一句话 |
|---|---|
| 商标的价值 | 使用场景（品牌销售、电商入驻）+ 防御性注册（防止抢注） |
| 申请在先原则 | 谁先申请商标就归谁，晚一两天可能就拿不到 |
| 商标 ≠ 公司名 | 公司名同地区可重名，商标全国唯一 |
| 商标 ≠ LOGO | 文字商标最重要，图形和声音也可以注册 |
| 通过率 | 全国仅50%~60%，接近一半可能被驳回 |
| ® 和 TM | ® 是已注册商标（拿到证才能用），TM国内不承认 |
| 注册周期 | 半年到一年 |
| 费用 | 自己270元/类，第三方约320元/类，顾问约570~770元/类 |
| 有效期 | 10年，无年费，到期续展 |
| 撤三风险 | 三年不使用可能被他人申请撤销，需留存使用证据 |

### 公司注册与商标的联动策略

1. **起公司名时，先查商标**——公司名可以重名，但商标不能，选一个同时能注册商标的名字

2. **公司注册后，立即申请商标**——拿到营业执照后第一时间去注册商标

3. **先提交商标申请，再宣传品牌**——防止被抢注

4. **注册核心类别+防御性类别**——至少3~4个类别

## 课后思考

### 思考题

1. 五险一金中，你认为哪个险种对你目前来说最重要？为什么？

2. 假设你给员工开税前工资1万元，每个月企业实际要支出多少钱？员工到手多少钱？中间的差额去哪了？

3. 公积金缴存比例选5%还是12%？各自的优劣势是什么？

4. 给员工减员的操作必须在什么时间之前完成？如果错过了这个时间怎么办？

5. 从公司注册到商标注册，你觉得应该怎么安排先后顺序，才能既高效又安全？
