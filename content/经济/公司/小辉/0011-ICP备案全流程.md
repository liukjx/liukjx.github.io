# 第十一课：ICP备案全流程

阶段二：注册实操第 11 课 / 共 11 课

> **本课目标**
> 学完本课，你将理解ICP备案是什么、为什么要做，并能够独立完成从注册云服务账号到提交备案申请的全流程操作。

## 一、ICP备案是什么？

**ICP备案**（Internet Content Provider 备案）是中国法律要求的网站行政审批制度。简单说，**任何在中国大陆上线的网站，都必须先做ICP备案**。

**通俗理解**就像开公司要办营业执照一样，网站上线也要办"网站身份证"——这就是ICP备案。

## 二、为什么要做ICP备案？

### 两个主要用途

| 用途 | 说明 |
|---|---|
| **1. 开通抖店教育培训类目** | 在抖店发布商品选"教育培训"类目时，系统要求补充行业资质，其中就包括ICP备案截图。这是目前最常见的需求。 |
| **2. 上线网站** | 以前更多是为了上线网站，现在大家基本不做网站了，主要需求就是抖店。 |

### 法律要求

- **ICP备案**：网站上线必需的行政审批，不备案不能上线

- **公安备案**：部分地区还要求到当地公安局做公安备案

- **等级保护（等保）**：涉及用户个人信息、交易信息的网站可能需要等保认证

## 三、备案前准备

开始前需要准备两样东西：

### 材料准备

- 公司营业执照副本照片（拍清晰）

- 法人本人在场（需要配合人脸识别）

- 身份证（法人身份证正反面）

### 账号准备

- 腾讯云/阿里云账号（推荐腾讯云）

- 已完成的实名认证（企业认证）

- 已注册的域名

- 至少一台云服务器（可买最便宜的）

## 四、注册云服务账号并认证（以腾讯云为例）

### 1. 注册登录

1. 打开腾讯云官网，点击右上角"登录"

2. 用手机微信扫码

3. 新用户会提示绑定注册账号，按指引完成注册

### 2. 企业认证

1. 点击"立即认证"

2. 选择"企业认证"，上传营业执照照片
- 图片要求不超过5MB，太大可用Windows画图工具缩小像素（如1920×1440）

3. 填写企业信息（推荐用天眼查查询公司信息复制粘贴）

4. 选择行业（不确定就选"其他"），通讯地址用注册地址

5. 认证方式选**"企业法人人脸识别"**（最快，即时完成）

6. 填写法人姓名和身份证号，手机扫码做人脸识别

7. 通过后即完成账号注册和实名认证

## 五、购买域名并实名

1. 回到腾讯云首页，搜索"域名注册"

2. 输入想要的域名名称（如 xiaohui-caishui），系统显示可用后缀及价格

3. 选择后缀：
| 后缀 | 通用性 | 价格（约） | 建议 |
|---|---|---|---|
| `.com` | 最强 | 60元/年 | 最推荐 |
| `.club` | 一般 | 12元/年 | 图便宜可选 |
| `.cc` | 一般 | 18元/年 | 图便宜可选 |

4. 选择1年即可，**务必开启"自动续费"**（防止到期后备案被注销）

5. 首次购买需创建域名信息模板：
- 点击"使用账号信息填写"（自动填入已认证的企业信息）
- 验证邮箱
- 再上传一次营业执照（要求55KB~2MB）
- 提交后等待注册局实名审核（可能需要3天左右）

6. 信息模板通过后，回去完成域名购买

7. 购买后域名状态从"注册中"→"审核中"→"正常"（约10分钟到1~3个工作日）

> **重要变化**
> 现在要求**先实名再购买域名**，与以前的流程相反，需要留出充足时间。

## 六、购买服务器

> **省钱技巧**
> 不要从"云服务器"入口直接买（最低约65元/月、600+元/年），而是从首页"最新活动"进入，找到"轻量应用服务器专场"，一年仅需**50元**。

- 只为了ICP备案的话，买最便宜的即可

- 如果有实际用途，根据需求选择配置

- 支付后等待服务器创建完成（约几分钟）

## 七、备案流程详细步骤

准备工作（账号、域名、服务器）全部就绪后，开始正式备案：

| 步骤 | 操作 | 注意事项 |
|---|---|---|
| 1. 进入备案 | 腾讯云 → "备案"→ "开始备案" | 选择备案省份（按公司注册地选） |
| 2. 主办单位信息 | 主办单位性质选"企业"，上传营业执照 | 系统自动识别企业名称和证件号码 |
| 3. 填写域名 | 输入已购买的域名 | 确保域名已实名认证通过 |
| 4. 服务器信息 | 系统自动带入已购买的云资源 | 确认IP信息正确 |
| 5. 上传负责人信息 | 上传法人身份证正反面照片 | 照片清晰、不反光 |
| 6. 网站名称 | 建议用公司全称或简称 | 去掉地区、行业、组织形式后的字号部分比较好通过 |
| 7. 人脸核验 | 在手机上完成人脸识别 | 法人本人在场 |
| 8. 网站备注 | 填写网站功能用途 | 建议写"某某公司的官网" |
| 9. 服务内容 | 选"单位门户网站" | 不要选错 |
| 10. 上传实名截图 | 上传域名实名截图 | 确保跟备案域名一致 |
| 11. 提交审核 | 检查信息无误后提交 | — |

## 八、审核流程

### 1. 腾讯云初审

- 客服会打电话核实信息

- 通过则进入下一步；有问题按要求修改后重新提交

- 新手首次提交通常会有一两个小问题，属于正常情况

### 2. 管局审核

- 腾讯云审核通过后，提交到工信部管局审核

- 管局审核期间会收到**短信核验通知**，包含验证码和链接

- 需要在PC端打开链接，输入验证码完成核验

- 继续等待审核结果

### 3. 审核通过

- 收到审核通过的通知短信

- 整个流程至少需要**一周**时间，建议提前规划

## 九、备案通过后查询截图

1. 打开搜索引擎，搜索"域名信息备案管理系统"

2. 找到右侧带"官方"标签的网站（工信部备案管理系统官网）

3. 输入单位名称点击搜索

4. 未通过的搜不到，通过后点击"详情"

5. 将该网页截图保存（这就是ICP备案截图）

6. 到抖店 → "资质中心"→ "行业资质"→ 编辑，上传ICP备案截图

7. 一般一个工作日左右审核通过

## 十、备案后的注意事项

| 事项 | 说明 |
|---|---|
| **变更备案** | 公司信息（名称、法人、地址等）变更后，需及时更新备案信息 |
| **注销备案** | 网站不再使用时，需主动注销备案，否则仍会承担法律责任 |
| **每年年审** | 每年需登录备案系统更新信息，确保备案持续有效 |
| **域名续费** | 域名到期未续费会导致备案被注销，务必开启自动续费 |
| **服务器续费** | 服务器到期未续费也会影响备案的有效性 |

## 十一、完整流程总览

> **ICP备案完整流程图**
> 准备材料（营业执照+法人）→ 注册腾讯云并企业认证 → 购买域名并实名（3天）→ 购买服务器（50元/年）→ 提交备案申请 → 腾讯云初审（电话核实）→ 短信核验 → 管局审核 → 审核通过 → 查询截图 → 上传抖店/上线网站

**整个流程至少预留一周时间，建议提前规划。**
