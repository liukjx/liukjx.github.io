# 第12课：财税基础知识深化

阶段三：财税合规第12课 / 共17课

> **本课目标**
> 在阶段一财税基础之上，深度掌握三大税种的实操规则、两条涉税红线的法律后果、小微企业优惠政策的落地方法，以及三张财务报表的核心指标与解读技巧。学完本课，你将具备基本的财税判断力，能看懂财务数据并做出正确的税务决策。

## 一、三大税种详解

公司经营离不开三个税种——增值税、企业所得税、个人所得税。一句话总结：**公司有收入→交增值税；公司有盈利→交企业所得税；公司给个人分钱→交个人所得税。**

### 1.1 增值税

增值税针对"增值部分"征税。例如：进一把吉他400元，卖出500元，增值100元，理论上只对100元交税。但在实务中，小规模纳税人和一般纳税人的处理方式不同。

#### 小规模纳税人 vs 一般纳税人对比

| 对比项 | 小规模纳税人 | 一般纳税人 |
|---|---|---|
| 计税方式 | 全部收入统一计税（不区分增值部分） | 只对增值部分交税（进销项抵扣） |
| 税率 | 优惠税率1%（疫情期间，原3%） | 6% / 9% / 13%（按行业分档） |
| 默认开票 | 普票 | 专票 |
| 能否抵扣 | 不能（不管普票还是专票） | 进项票可抵扣销项票 |
| 能否开专票 | 可以（去税务局代开） | 默认开专票 |

**选择建议**根据业务需求判断：上下游是否要求开专票？计算哪种税率更优惠？如果客户需要专票抵扣，一般纳税人更有优势；如果客户多为个人消费者，小规模纳税人税负更低。

### 1.2 企业所得税

针对企业盈利（收入 - 成本 - 费用）征收。小微企业享有显著优惠税率。

#### 企业所得税阶梯税率表

| 年盈利区间 | 适用税率 | 说明 |
|---|---|---|
| 100万以内 | **2.5%** | 小微企业优惠，实际税负极低 |
| 100万 ~ 300万 | **10%** | 小微企业优惠，税率适中 |
| 300万以上 | **25%** | 统一税率，不再享受小微企业优惠 |

> **300万是关键分界线**
> 年盈利超过300万（哪怕多出301元），就不再是小微企业，全额按25%征收。假设年利润301万，比300万多1万，但要多交约45万的企业所得税。合理控制利润在300万以内非常重要。

**其他优惠：**

- **高新技术企业：**企业所得税从25%降至15%，且研发费用可按175%加计扣除（增加成本→降低利润→少交税）

- **海南等特定地区：**企业所得税可享受15%优惠税率

### 1.3 个人所得税

公司给个人分钱就要交个税。个税的代扣代缴义务在公司，所以老板必须关心。

#### 三种个税类型对比

| 类型 | 适用场景 | 税率 | 计算方式 |
|---|---|---|---|
| 个人综合所得 | 工资、奖金、年终奖 | 3% ~ 45%（7档阶梯） | 按月预扣、年度汇算清缴 |
| 个人经营所得 | 个体户、个人独资企业经营者 | 5% ~ 35%（5档阶梯） | 按年计算、按月/季预缴 |
| 其他所得 | 股东分红、股权转让、房屋租赁 | 20%（统一税率） | 分类征收 |

**需要交个税的场景：**发工资、发奖金、股东分红、股权转让。

**不用交个税的场景：**

1. 员工垫付→凭发票报销（属于业务报销，不是个人收入）

2. 公司跟个人合作→个人去税局代开发票（个人已在开票时完税）

3. 公司借款给个人→必须还回，不超过一个财年，且不能借给股东或员工长期占用

## 二、两条涉税红线

> **郑重提醒**
> 以下两条红线触碰后可能面临刑事处罚，请务必重视。

### 2.1 红线一：逃税罪（刑法第201条）

**处罚标准：**数额巨大且占应纳税额30%以上的，处3年以上7年以下有期徒刑。

#### 三种典型逃税行为

| 行为类型 | 具体表现 | 风险等级 |
|---|---|---|
| **私户收款不入账** | 用个人银行卡收业务款，不进公司账、不做账报税 | 极高 |
| **一般户收款不入账** | 基本户报税了，一般户（其他银行账户）悄悄收款不记账 | 极高 |
| **买发票增加成本** | 花钱购买发票来降低利润，既是逃税又是虚开 | 极高（双重违法） |

> **金税系统预警**
> 金税三期/四期后，税局可以查到所有平台的收入（电商、个人银行卡、微信、支付宝等），私户收款已经不再安全。任何异常资金流都会触发系统预警。

### 2.2 红线二：虚开增值税专用发票罪（刑法第205条）

**处罚标准：**情节最高可判**无期徒刑**，比逃税罪更严重。

#### 四种虚开类型

| 类型 | 举例 |
|---|---|
| **为他人虚开** | 朋友缺成本票，你帮他开具发票——这是虚开 |
| **为自己虚开** | 注册个独企业给自己的公司开具发票 |
| **让他人为自己虚开** | 让别人给你开不是真实业务的发票 |
| **介绍他人虚开** | 做中间人介绍虚开，也算共同违法 |

#### 三流合一标准（证明业务真实的最低要求）

| 三流 | 对应证据 |
|---|---|
| **资金流** | 银行转账记录 |
| **发票流** | 开出去和收到的发票 |
| **业务流** | 合同 + 物流/业务成果/沟通记录 |

> **判断标准**
> 核心标准只有一个：**是否有真实业务发生**。三流合一只是形式合规的最低要求，不代表实质合规。但如果连三流合一都做不到，就更容易被查出问题。

## 三、小微企业优惠政策

### 3.1 增值税优惠

- 小规模纳税人税率优惠至**1%**（原3%）

- **每季度开票不超过45万**→ 免征增值税

- 超过45万则全额交税（不是只交超出部分）

**执行建议：**

- 控制开票时间——季度快到45万时，跟客户协商推迟到下季度开票

- 如果票量足够大——**分拆公司**（如将"消防设备销售+安装"拆成销售公司和安装公司，每家独立享受45万免征额度）

### 3.2 企业所得税优惠

年盈利300万以内为小微企业，享受2.5% / 10%的优惠税率。

**增加成本的合法方式（降低利润→少交税）：**

- 以公司名义花钱（买电脑、招待客户、外出学习等），要求提供发票

- 给自己和员工发工资、发奖金

- 以公司名义买车（算公司资产，可折旧）

- 所有花钱的地方都要求发票

### 3.3 个人所得税优惠

| 优惠政策 | 具体内容 | 每年节省（估算） |
|---|---|---|
| 年免征额 | 每月5000元以内免税，年共6万 | 基础免征 |
| 专项附加扣除（6项） | 子女教育、继续教育、住房贷款利息、住房租金、赡养老人、大病医疗 | 每项约1000~2000元/月 |
| 社保和公积金免税 | 公积金按12%缴纳，缴纳后可提取（以买房/装修/租房名义） | 约数万元/年 |
| 年终奖单独计税 | 一年只能用一次，与工资分别走独立阶梯税率通道 | 显著降低税负 |

> **综合测算**
> 免征额 + 专项扣除 + 社保公积金，一个人年入十二三万以内基本不用交个税。

## 四、三张财务报表

### 4.1 三表概览

| 报表 | 反映什么 | 时间维度 | 核心指标 |
|---|---|---|---|
| **资产负债表** | 公司有多少钱、欠多少、值多少 | 某一时间点（快照） | 资产、负债、股东权益 |
| **利润表** | 公司赚了多少、花了多少、盈利还是亏损 | 某一时间段（如一个月/一年） | 收入、费用、利润 |
| **现金流量表** | 现金流入流出情况 | 某一时间段 | 经营/投资/筹资活动的现金流 |

### 4.2 资产负债表

**核心公式：**`资产 = 负债 + 股东权益（净资产）`

- **资产总计：**公司有多少钱（流动资产 + 非流动资产）

- **负债合计：**欠多少钱

- **股东权益：**公司真正值多少钱（资产 - 负债）

> **老板必看指标**
> 总资产大不代表实力强（如恒大资产上万亿但负债率90%以上）。要看**净资产（股东权益）**，这才是公司真正属于自己的部分。

### 4.3 利润表

**核心逻辑：**

```
营业利润 = 收入 - 费用
利润总额 = 营业利润 + 营业外收支
净利润 = 利润总额 - 所得税
```

### 4.4 现金流量表

分为三部分：

- **经营活动：**主营业务赚的钱——最重要的现金流来源

- **投资活动：**投资收益/亏损（如买理财、卖资产）

- **筹资活动：**融资/分红/还贷款

> **老板必看指标**
> 重点看现金流是否健康。一个关键问题：**在没有任何收入的情况下，公司账上的现金能支撑几个月？**对小微企业尤其重要——很多公司有利润但没现金，死在现金流断裂上。

### 4.5 老板必须关注的指标汇总

| 财务指标 | 从哪张表看 | 健康标准 |
|---|---|---|
| 净资产（股东权益） | 资产负债表 | 正数且持续增长 |
| 净利润 | 利润表 | 正数，利润率不低于行业平均水平 |
| 经营活动现金流 | 现金流量表 | 正数，且大于净利润（含金量高） |
| 资产负债率 | 资产负债表 | 一般不超过60% |
| 现金储备月数 | 现金流量表 | 至少能支撑3-6个月 |

### 课后思考题

1. 你目前公司的年利润在哪个区间？是否有办法将利润控制在300万以内以享受小微企业优惠税率？

2. 公司是否存在私户收款或买发票的行为？如果有，请立即停止并咨询专业财务顾问进行合规整改。

3. 上个月公司的净利润和经营活动现金流分别是多少？两者相差大吗？如果净利润为正但现金流为负，问题出在哪里？

4. 你公司目前的增值税纳税人身份是哪种？上下游的发票需求是怎样的？是否需要更换纳税人身份？

5. 查看最近一期的资产负债表，公司的资产负债率是多少？净资产在增长还是减少？
