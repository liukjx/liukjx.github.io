# 第14课：股权架构与金税四期

阶段三：财税合规第14课 / 共17课

> **本课目标**
> 掌握公司控制权的5个层面和4种工具、理解各种企业类型的区别及适用场景、深刻认识金税三期到四期的升级变化及其对企业的影响。学完本课，你将能设计合理的股权架构保护控制权，并在金税四期环境下做到合规经营。

## 一、搭建股权架构——把控公司控制权

### 反面案例警示

| 案例 | 问题 | 教训 |
|---|---|---|
| **ofo戴威** | 滴滴和阿里都有一票否决权，想卖公司卖不了，陷入僵局 | 控制权分散导致决策僵局 |
| **西少爷** | 三人合伙（40%/30%/30%），没有绝对老大，闹矛盾差点分家 | 股权平均是灾难 |
| **李子柒** | 与MCN合资公司中MCN占51%，李子柒本人只占49%，无法控制商标 | 创始人失去品牌控制权 |

### 公司控制权的5个层面

| 层面 | 说明 | 控制要点 |
|---|---|---|
| **股权** | 最底层控制权，决定股东会投票权 | 持股比例决定话语权 |
| **董事会** | 占多数席位，很多事项出董事会决议即可 | 控制董事会席位多数 |
| **法定代表人** | 公司对外签字代表 | 一般由大股东/实际控制人担任 |
| **公章** | 盖过章的文件代表公司决策意志 | 建议老板自己保管 |
| **银行UKey** | 谁拿着谁控制资金转账 | 掌握资金进出权限 |

### 股权控制权的4个关键数字

| 持股比例 | 权力 | 能做/不能做什么 |
|---|---|---|
| **67%（2/3以上）** | 绝对控制权 | 任何事都能决定（注销、并购、增资等重大事项） |
| **51%（过半数）** | 相对控制权 | 一般事项过半通过即可 |
| **34%（1/3以上）** | 一票否决权 | 别人凑不够2/3，重大事项你一票就能否决 |
| **10%** | 小股东掀桌权 | 可发起临时股东会、请求公司清算 |

### 四种控制权工具对比

| 工具 | 原理 | 适用场景 | 优点 | 缺点 |
|---|---|---|---|---|
| **一致行动协议** | 多个小股东约定投票方向一致 | 股权较分散时联合其他股东 | 操作简单，灵活 | 依赖合同约束力，可能违约 |
| **有限合伙持股平台** | 创始人担任GP掌握决策权，员工/投资人担任LP享分红权 | 员工股权激励、投资人入股 | 控制权集中，税务灵活 | 设立成本较高 |
| **AB股制度** | 同股不同权：A股1票，B股多票（如10票或20票） | 多轮融资后创始人仍想控制公司 | 以少量股权控制公司 | 国内上市不支持，需搭建VIE |
| **董事会控制** | 控制多数董事会席位 | 与股权控制配合使用 | 日常决策效率高 | 重大事项仍需股东会 |

### 分红权与决策权分离

**方式1——代持协议：**被代持方只享有分红权，控制权归代持方。

**方式2——有限合伙持股平台：**

- 创始人担任**GP**（普通合伙人/执行事务合伙人）→ 掌握全部决策权

- 合伙人和核心员工担任**LP**（有限合伙人）→ 只享有分红收益权

- 投资人直接持有业务公司股份

```
创始人（GP）          员工/合伙人（LP）
          │                      │
          └──────────┬───────────┘
                     │
              持股平台（有限合伙）
                     │
                     ▼
               业务公司主体
```

## 二、各种企业类型区别及用途

### 六大企业类型全面对比

| 对比项 | 有限责任公司 | 股份有限公司 | 有限合伙 | 个人独资企业 | 个体工商户 |
|---|---|---|---|---|---|
| 股东人数 | 1~50人 | 2~200人 | 2~50人 | 仅1人 | 仅1人 |
| 责任类型 | 有限责任 | 有限责任 | GP无限/LP有限 | 无限责任 | 无限责任 |
| 企业所得税 | 交（2.5%~25%） | 交（25%） | 不交（穿透征税） | 不交 | 不交 |
| 个人所得税 | 分红20%/工资3%~45% | 分红20%/工资3%~45% | 经营所得5%~35% | 经营所得5%~35% | 经营所得5%~35% |
| 是否可核定征收 | 极难 | 极难 | 可以 | 可以（政策收紧） | 相对容易 |
| 入驻电商平台 | 无限制 | 无限制 | 有限制 | 基本无限制 | 有较多限制 |
| 可招聘员工 | 无限制 | 无限制 | 无限制 | 无限制 | 不超过8人 |
| 是否可转让 | 可以 | 可以（上市后可交易） | 可以 | 可以 | 不可以 |
| 开分公司/子公司 | 都可以 | 都可以 | 不可以 | 不可以 | 不可以 |

### 各类企业的主要用途

| 企业类型 | 主要用途 |
|---|---|
| **有限责任公司** | 主营业主体，做业务、签合同、招聘员工、融资 |
| **股份有限公司** | 准备上市的公司主体，股权结构更灵活 |
| **有限合伙企业** | 持股平台（股权激励、投资基金），GP控制决策权，LP分享收益 |
| **个人独资企业** | 核定征收、咨询服务、居间费处理 |
| **个体工商户** | 核定征收最方便，适合小规模经营、成本票处理 |

> **实操建议**
> 可以各注册一个，需要用到哪个场景就用哪个。有限公司做主业务，个体户/个独做核定征收和成本票处理。有限合伙做持股平台——多种企业类型组合使用，实现最优的税务和法务架构。

## 三、金税三期和四期的威力

### 3.1 金税三期（2005年启动，历时10年打造）

- 统一全国税务核心征管系统

- 覆盖所有税种、主要工作环节、各级税务机关

- 实现税务系统**内部数据全国联网**

### 3.2 金税四期——"以数治税"的升级

| 对比维度 | 金税三期 | 金税四期 |
|---|---|---|
| 核心理念 | 以票管税 | **以数治税** |
| 数据范围 | 税务系统内部 | 各部委、人民银行、商业银行信息共享 |
| 监控对象 | 企业纳税情况 | 企业全方位数据（银行流水、社保、公安等） |
| 监控方式 | 单向税务检查 | **全方位立体化监控** |
| 技术手段 | 联网对比 | 大数据分析、人工智能预警 |

### 3.3 金税四期对企业的影响

**直接影响：**

- 私户收款、隐匿收入将无所遁形——税局可查个人银行卡流水

- 社保缴纳情况透明化——未足额缴纳社保会被发现

- 发票异常自动预警——虚开、买发票行为极易被发现

- 银行与税务数据打通——资金流向异常将触发稽查

### 3.4 合规应对策略

| 应对领域 | 具体措施 |
|---|---|
| **收入管理** | 所有收入进公户，不做私户收款、不设账外账 |
| **成本管理** | 所有支出获取合规发票，不买票、不虚开 |
| **社保管理** | 按规定缴纳社保，通过工资拆分、丰富用工模式合理降本 |
| **税务筹划** | 用好小微企业优惠政策、核定征收等合法方式 |
| **专业支持** | 聘请专业财务顾问，定期进行税务健康检查 |

> **核心结论**
> 金税三期/四期后，所有平台数据都已打通，依法纳税、合规经营是企业的唯一选择。在合规的前提下，用好小微企业的优惠政策，才是正确的税务筹划方向。

### 课后思考题

1. 你目前在公司中的持股比例是多少？是否达到了绝对控制权或一票否决权的门槛？

2. 如果你的公司引入了合伙人或投资人，你会用哪种工具来保护控制权？有限合伙持股平台还是一致行动协议？

3. 你的公司目前是否有使用个体户或个独处理成本票？如果有，操作是否合规？

4. 你过去有没有使用私户收款的行为？在金税四期环境下，应该如何调整？

5. 你是否了解公司目前的社保缴纳情况？是否做到了全员足额缴纳？如果成本压力大，可以采用哪些合法方式优化？
