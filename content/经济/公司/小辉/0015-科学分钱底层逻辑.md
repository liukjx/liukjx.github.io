# 第15课：科学分钱底层逻辑

阶段四：团队激励第15课 / 共17课

> **本课目标**
> 深入理解科学分钱的底层逻辑：从老板的三大痛点出发，理解"分钱不是分存量而是激发增量"，掌握科学分钱三角循环模型，识别5种错误分钱方式，学习华为"激发和控制欲望"的核心原理。学完本课，你将建立起科学分钱的思维框架，为后续具体的激励方法打下理论基础。

## 一、老板的三大痛点

### 痛点1：老板很忙

忙是老板的常态，但忙什么却大有不同。用"重要/紧急四象限"来分析：

|  | 紧急 | 不紧急 |
|---|---|---|
| **重要** | 危机处理、客户投诉、交期临近 | 制定战略、搭建团队、建立机制 |
| **不重要** | 日常琐事、打扰性会议 | 刷手机、无意义的社交 |

小米CEO雷军说过：**"不要用战术上的勤奋掩盖战略上的懒惰。"**很多创业者业务不大却比上市公司CEO还忙，这是自我感动。创业者需要抽身思考：制定什么样的机制能让员工更努力工作，把公司的事当成自己的事来做。

### 痛点2：老板心很累

- 各部门之间不配合，出了事没人担责

- 员工不关心成本，只有老板关心

- 例：服装连锁店员工宁愿花30元发闪送调货，也不愿骑车10分钟自取，因为成本跟他们无关

### 痛点3：老板很孤独

- 员工不理解、家人不理解，连合伙人也不能完全交心

- 很多所谓的"合伙人"无非是拿的薪水高一些、管的事多一些，并不是和老板完全一条心

## 二、火车与动车的故事

**核心隐喻**以前火车最高时速100公里——动力全在车头，后面车厢只负责载人。现在的动车组时速可达300-400公里——**每节车厢都是独立的动力系统**，合起来一起使劲。

**企业经营的启示：**如果动力系统全靠老板一个人带动，效率不会太高；如果每个部门、每个员工都干劲十足，公司就会像动车组一样高效。老板要做的不是当那个最大的火车头，而是想办法让每一节"车厢"都有自驱力。

## 三、科学分钱三角循环

### 核心模型

```
员工努力干 → 公司能赚钱 → 员工分到钱 → （循环回来）
```

### 三条链条缺一不可

| 链条 | 断裂后果 |
|---|---|
| 员工努力干 ←→ 公司能赚钱 | 努力方向不对，公司无法获利 |
| 公司能赚钱 ←→ 员工分到钱 | 公司赚钱但员工分不到 → 失去干劲 |
| 员工分到钱 ←→ 员工努力干 | 分钱不合理 → 员工选择单干或跳槽 |

> **老板的使命**
> 老板的使命就是搭建这个循环。因为员工无法单独在市场上赚到钱，才需要公司这个组织架构。如果体系搭建不高效，员工要么单干要么跳槽。

## 四、5种错误分钱方式

| 错误方式 | 具体问题 | 正确做法 |
|---|---|---|
| **①固定工资+固定奖金** | 员工努力干→公司赚钱→但拿到的钱是固定的→循环断裂 | 工资+浮动的绩效/提成，让回报与贡献挂钩 |
| **②留人式涨工资** | 员工提离职→老板涨工资留下→其他员工也学会"用离职换涨薪" | 建立主动的薪酬调整机制，不要被动应对 |
| **③普涨（全员涨薪）** | 大锅饭、平均主义，干好干坏都涨→员工只做到"不被开掉"的最低标准 | 差异化激励，让奋斗者获得更多回报 |
| **④无条件一次性授予股份** | 拿到股份后立刻躺平，搭便车心理 | 设置得权期（vesting）和退出机制 |
| **⑤溢价商品按收入提成** | 涉及折扣的商品按收入提成→员工薄利多销但公司可能亏钱 | 溢价类商品按毛利提成 |
| **⑥前后端分离各算各的绩效** | 前端销售循环起来了，中后台没有循环→前端也无法很好赚钱 | 前后端绩效打通，中后台也要有激励 |

## 五、底层逻辑来自华为

### 华为的核心经验

2017年华为在上海开了一次四天会议，结论是：**坚持方向大致正确，组织充满活力。**

组织由个人组成，要让组织有活力就要让个人充满活力。

### 激励的基本逻辑链

```
欲望 → 动机 → 动力 → 动作 → 好的结果 → 新循环
```

科学分钱的底层逻辑就是**激发和控制欲望**。

### 欲望的来源

1. **生存状态不圆满：**人永远在追求圆满但始终无法达到，这种"有目标但还没实现"的状态就形成欲望

2. **对比产生不圆满：**同样月薪1万，在四五线小城是佼佼者，在北京30多岁拖家带口就不够糊口

### 如何激发欲望：构筑利益差

**外部利益差：**与同行和市场上其他企业相比，提供更有竞争力的利益分配机制，吸引优秀人才。

**内部利益差：**以奋斗者为本，让奋斗者获得超过普通劳动者的回报，拉开分配差距。

### 多维度构筑利益差——分利、分权、分名

| 维度 | 方式 | 华为案例 | 企业可以怎么做 |
|---|---|---|---|
| **分利** | 薪酬分享机制 | 任正非股份不到1%，利润几乎全分掉 | 设计提成、奖金、分红等多种分配方式 |
| **分权** | 权力分享机制 | 轮值CEO、轮值董事长制度 | 授权管理者决策权，设立见习制度 |
| **分名** | 荣誉分享机制 | 大量荣誉奖项、精神激励 | 排行榜、颁奖仪式、给父母寄感谢信 |

### 如何控制欲望：两个核心原则

| 原则 | 含义 | 企业落地 |
|---|---|---|
| **原则1：以企业经营目标增长为导向** | 激励机制必须围绕如何促进公司业务增长 | 如果业务没有增长，激励机制大概率有问题（大锅饭/利益板结） |
| **原则2：以员工持续奋斗为导向** | 激励不是一次性，而是持续激发 | 华为的"三高"：高绩效（牵引）+ 高激励（动力）+ 高压力（淘汰），一个都不能少 |

### 课后思考题

1. 你目前属于四象限中的哪一类？是在做重要的事还是只在忙紧急的事？

2. 你的公司目前属于"火车"模式还是"动车"模式？有多少员工是自发地在推动业务前进？

3. 你目前的薪酬分配方式中，有没有本课提到的6种错误方式？具体是哪几种？

4. 你公司的内部利益差足够大吗？干得最好的员工和中等水平的员工收入差距是几倍？

5. 你在"分利、分权、分名"三个维度上做得如何？哪个维度最需要加强？
