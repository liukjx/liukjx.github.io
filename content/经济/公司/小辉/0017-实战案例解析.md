# 第17课：实战案例解析

阶段四：团队激励第17课 / 共17课

> **本课目标**
> 通过完整的实战案例，掌握科学分钱激励系统的四步搭建方法，并学习三个不同行业（互联网大厂、连锁培训机构、游戏公司）的真实薪酬与股权方案设计。学完本课，你将能根据自己企业的实际情况，设计出可落地的激励方案。

## 一、四步搭建科学分钱激励系统

### 步骤1：明确目标

- 目标要**简单清晰**，越简单越容易达成

- **一次只定一个目标**（如提升销售业绩、留住优秀员工、提高部门协同）

- 建立正反馈循环，让大家都能从中获益

### 步骤2：选择方法

| 方法 | 核心目标 | 适用对象 |
|---|---|---|
| **提成** | 增加销售业绩 | 销售、渠道、商务等业务岗位 |
| **绩效** | 支持公司长远发展 | 产品、技术、运营、职能等中后台岗位 |
| **晋升** | 培养内部管理人才 | 有管理潜力的骨干员工 |
| **调岗** | 培养复合型人才 | 寻求职业突破的员工 |
| **股份** | 帮老板分担压力，建立主人翁意识 | 核心合伙人、关键岗位高管 |
| **孵化** | 参与快速成长员工的收益，避免成为竞争对手 | 有创业潜质的优秀员工 |

### 步骤3：快速试错（MVP模型）

- 先找一个团队或员工试点

- 试错了影响范围小，试对了让他们帮你宣传

- 小范围验证后再全面推广

### 步骤4：全面推广

- 让实施激励的对象帮你宣传（分享会、交流讨论）

- 做好宣传节奏，避免员工觉得"老板又来克扣工资"

- 用数据说话：展示试点团队的业绩增长和收入提升

### 四步搭建全流程

```
明确目标 → 选择方法 → 快速试错 → 全面推广
                              ↓
                        一个团队/员工试点
                              ↓
                     试对了 → 分享宣传 → 全面推广
                     试错了 → 调整方案 → 再次试点
```

## 二、互联网大厂案例——数字化薪酬及股权方案

### 企业背景

某一线互联网大厂，科技型公司，需要创意和协作的组织。使用了飞书绩效作为数字化绩效工具。

### 评估流程

```
自评 → 邀请360度评估（8-16人）→ 上级纠偏 → 上上级纠偏 → 最终绩效结果
```

### 评估维度

| 维度 | 权重 | 说明 |
|---|---|---|
| 业绩 | 主要导向 | 核心考核指标，权重最高 |
| 价值观 | 辅助 | 是否符合公司文化价值观 |
| 投入度 | 辅助 | 工作态度和投入程度 |

### 绩效分档与激励

| 绩效档位 | 定义 | 绩效系数 |
|---|---|---|
| S | 杰出（超出预期） | 2.0 |
| A | 优秀 | 1.5 |
| B | 符合预期 | 1.0 |
| C | 需要改进 | 0.5 |
| D | 不合格 | 0 |

> **防作弊机制**
> 系统自动检测分布是否异常（如某部门全评S会触发提醒）。确保绩效分布合理，避免"老好人"打分。

### 数字化股权工具

推荐工具：**易餐**、**股权**等数字化平台。

- 让员工通过手机APP看到自己有多少股份、何时成熟、能变现多少

- 变虚为实，从模糊印象变成可量化的可视化资产

- 大幅提升股权激励的感知度

## 三、连锁培训机构案例——薪酬及股权方案

### 企业背景

一家连锁培训机构，有多个校区，需要激励销售（课程顾问）、老师（教学交付）和校长（校区经营）三类角色。

### 销售（课程顾问）薪酬方案

| 项目 | 标准 |
|---|---|
| 基础工资 | 4000~5000元 |
| 提成 | 订单毛利 × 10%~50% |
| 新客/老客区分 | 新客提成是老客的2倍 |
| 10万以上大单 | 提成比例直接翻倍（二八原则） |
| 重点产品 | 额外补贴10% |

### 老师薪酬方案

| 项目 | 标准 |
|---|---|
| 基础工资 | 4000~5000元 |
| 课时费 | 100~300元/节（按课程难度+学员评分动态调节） |
| 试听课 | 额外提成（试听课是转化关键环节） |
| 转介绍 | 老师带来新客户也给提成 |

### 校长薪酬方案

| 项目 | 标准 |
|---|---|
| 基础工资 | 高于普通员工 |
| 绩效 | 经营业绩 × 8% + 业绩增幅 × 12% |
| 经营业绩 | 现金流收入 × 50% + 净利润 × 50% |

> **设计亮点**
> 校长绩效既考核现金流也考核真实利润，同时激励持续增长。业绩增幅的权重（12%）高于绝对业绩的权重（8%），鼓励校长不断超越自己。

### 分校股权方案

| 参与方 | 持股比例 | 说明 |
|---|---|---|
| 总部 | ≥50% | 保持相对控制权 |
| 校长 | 20%~30% | 最关键的角色，给最大比例 |
| 核心骨干 | 10%~20% | 所有人加起来 |

#### 三个核心要点

1. **过试用期再谈股份**（3-6个月，让校长看到公司是赚钱的）

2. **股份必须花钱买**，越早买越便宜

3. **每季度分红一次**，及时激励，让股权价值可感知

## 四、游戏公司案例——股权激励方案

### 企业背景与痛点

- 4个合伙人，曾口头承诺股份但未登记

- 合伙人经常变动

- 20名员工未承诺但希望得到激励

### 创始人核心诉求

1. 牢牢掌握公司控制权

2. 股份不一次性分完（4年得权期）

3. 合伙人要实名登记（安全感）

4. 员工满4年后再登记

5. 员工离职后低成本收回股份

### 方案架构

```
创始人（A轮投资人）
                   │
                   ▼
             游戏公司主体
                   │
          ┌────────┴────────┐
          │                 │
          ▼                 ▼
    持股平台          （直接持股）
  （有限合伙）
     │
  ┌──┴──┐
  │     │
  ▼     ▼
GP    LP（合伙人+员工）
（创始人）  （只有分红权）
（控制决策权）
```

### 关键条款设计

| 条款 | 合伙人 | 员工 |
|---|---|---|
| **得权期** | 4年，每年登记25% | 4年，成熟后一次性登记 |
| **当下登记** | 立即登记一部分（如10%） | 不登记，由创始人代持 |
| **退出回购价（有融资）** | 融资估值×0.1（打一折） | 公司定价直接赎回 |
| **退出回购价（无融资）** | 公司净资产÷总股数 | 同上 |
| **违约情形** | 协议约定 | 违反竞业禁止、造成重大损失等 |

> **方案亮点**
> ①用持股平台（有限合伙）将控制权集中在创始人手中；②合伙人实名登记给安全感，但设得权期防止躺平；③员工4年成熟后才登记，降低管理成本；④退出回购价打一折，既保留退出通道又避免套现过早。

## 总结：科学分钱核心要点回顾

| 核心维度 | 要点 |
|---|---|
| **底层逻辑** | 激发和控制欲望，构筑外部和内部利益差 |
| **六大方法** | 提成、绩效、晋升、调岗、股权、孵化——不同场景选不同方法 |
| **实操原则** | 利益差要大、多维构建（名权利）、导向经营增长、导向持续奋斗 |
| **搭建系统** | 明确目标→选方法→快速试错→全面推广 |
| **股权关键** | 得权期（金手铐）+ 退出机制 + 持股平台 + 及时分钱 |
