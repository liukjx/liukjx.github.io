第26课 · 第6周
# 在母语中撒娇

开智学堂·认知写作学第四期 | 第6章：修辞感觉——告白与提问

## 一、踌躇（求索）——与没收相对的极端

### 1.1 两种极端的语言反应

当你心情不好的时候，会产生两种极端的语言行为：

| 极端 | 行为 | 对应修辞 |
|---|---|---|
| **内敛** | 在纸上打一大堆省略号，写谁都看不懂的符号 | 没收（默说）——意在言中 |
| **外放** | 失恋后一口气写上几百个"王八蛋"，从不同角度去写 | **踌躇（求索）**——在母语中撒娇 |

没收是沉默不语，踌躇是无止境地使用大量词汇。这两种修辞手法构成了对话的两个极端。

### 1.2 什么是"在母语中撒娇"？

佐藤信夫对"踌躇"的定义是：罗列了一大堆词语，但总找不到那个准确的词语，因此显得非常啰嗦。传统的中国和西方修辞学书中都没有这一种修辞手法，因为传统修辞学是基于归纳，而不是逻辑推演体系。佐藤信夫是基于人类对话的现象推演出修辞认识的第二个层级。

为什么叫"在母语中撒娇"？

> 人类所依赖的语言，依赖的是母语。假如原本表达的是不好的情绪——比如失恋，你写成二三十个"王八蛋"——结果你发现，这变成一个饶舌歌手在唱歌了，有了一点幽默感。这种略带幽默的悲哀，相当于你在和你的母语撒娇，而不是和那个已经离开你的男朋友在撒娇。

### 1.3 认知科学基础：4×5原理

认知科学有一个4×5原理：人类写作和大脑遵从认知负荷最小的路径——简单反应词加不超过4个主块、不超过5个受损程式。在情绪正常时，你是一个理性的人，大脑会如此工作。

但当你失恋、情绪极端时，你的思维和语言发生了很大变化，就需要另一种表现方法。这就是踌躇的价值所在。

## 二、踌躇的两种路径

### 2.1 近义积累——太宰治《狂言之神》

太宰治的巅峰之作《狂言之神》中，描写一个人准备自杀前的心态——他写了一口气20个词汇：

> "没有阴郁也没有荒凉，没有孤独的极点也没有智慧的尽头，没有狂乱也没有愚蠢的感觉，没有嚎啕大哭也没有闷闷不乐，没有严肃也没有恐惧，没有形态也没有愤怒，没有看穿一切也没有秋风的凉意，没有和平也没有后悔，没有沉思也没有打算，没有爱也没有救赎。"

这是一流作家的写法。对比二三流作家的写法（冰心、朱自清等）——他们遵从4×5原理，写14个形容词就打住了。太宰治一口气写了20个，远远违背了4×5原理，但正是这种"违反规则"恰好表达了自杀前极度复杂的心态。

要注意的是：太宰治的20个词汇没有一个直接跟自杀或自杀相关的意象挂钩。二流作家会写"尖刀、毒药"等暗示自杀的意象；三流作家根本不知道4×5原理的存在；只有超一流作家像太宰治，一口气堆砌20个看似无关的词汇，围绕一个对话情景，情绪摇摆来摇摆去——这就是踌躇的最佳体现。

### 2.2 模糊表达——鲁迅《阿Q正传》

模糊表达是踌躇的高阶运用手法。人类大脑偏好清晰，倾向于给一切事物赋予因果关系，表达的更清晰。但一流作家会故意模糊化。

黑川雅之的模糊时钟设计：一般的时钟追求精确，有12个刻度；但黑川雅之故意把时钟做成模糊的——只有4个或2个刻度，甚至用磨砂玻璃让你看不清楚。这样的作品击中你的自主心智，让你秒懂。

在文学中的运用——川端康成《雪国》的经典翻译辨析：

| 翻译版本 | 内容 | 评价 |
|---|---|---|
| **一般翻译** | "岛村感到狼狈不堪" | 表达清晰明确，但破坏了意境 |
| **川端康成原文** | "岛村有些狼狈不堪" | "有些"二字把踌躇、暧昧的意境保留了下来 |

很多翻译家不懂认知科学，试图把意思表达得更清晰准确——但这种清晰准确反而是一种失误，把伟大作家精微描绘的男女之间的暧昧关系破坏掉了。

**鲁迅在《阿Q正传》**中大量使用"有些"，粗略统计在60处以上。最经典的阿Q精神描述段落中连续使用三处"有些"：

> "他立刻转败为胜。他举起自己的右手，用力的在自己脸上连连打了两个巴掌，热刺刺的有些痛。打完之后，阿Q便心平气和起来，似乎打的是自己，被打的是另一个自己，不久也仿佛是自己打了别人一般，虽然还有些热刺刺，心满意足的得胜的躺下了。"

如果把"有些"删掉，变成"非常痛"，阿Q精神的那种自我欺骗、自我麻醉的感觉就大打折扣了。

### 模糊表达的使用原则

模糊表达不是让你的文章变得啰嗦。它的使用场景是：**当你想表达一种模糊的意态、忧伤、暧昧的情绪时**，才使用这种手法。如果表达明确的意思，仍然要用清晰的语言。鲁迅的"有些"用得恰到好处，正是因为阿Q正传整体上是在表达一种反讽和国民性的暧昧。

## 三、卷语（视角切换）——对话的高级形态

### 3.1 什么是卷语

写作是一个人和另一个人的对话。卷语描述的现象是：对话时，你的视角不断在自己和对方之间切换。伟大的作家能不断转换视角，从"我方立场"切换到"对方视角"，形成一种自然的、让读者意识不到切换的语言。

佐藤信夫甚至认为：卷语是人类非常重要的一种思维方式，有可能比隐喻更重要。

### 3.2 经典案例："阿夏的头更加沉重了"

> "阿夏的头更加沉重了。"

这句话为什么是卷语？因为在此之前，一直是站在另一个对话者（利雄）的角度。阿夏没有回答，在这一句时，视角突然切换到了阿夏——读者感知到阿夏的头放在利雄手上，他睡着了，头砸在手上，变得更沉重了。

从阿夏的视角 → 切换到利雄的视角 → 微妙的视角切换，形成同理心的自然转换。这就是一流作家对视角切换的驾轻就熟。

### 3.3 黑帮老大敬酒

> "真的不能再喝了吗？""不能喝。""喝了吧，趁我还在好言相劝。"

这一段视角的切换非常精彩：

- "真的不能再喝了吗？"——视角在被劝酒的人身上

- "不能喝。"——被劝酒者回答

- "喝了吧，趁我还在好言相劝。"——视角切换到了劝酒的黑帮老大身上，从关心变成了威胁

这种语言的切换非常自然，一流作家的切换甚至让你意识不到已经产生了切换。

## 四、侧写（侧面描写）——视角切换的变体

### 4.1 《陌上桑》的侧面描写

中国古代诗歌中大量使用侧写（侧面描写）。最经典的例子是汉乐府《陌上桑》对美女秦罗敷的描写：

> "行者见罗敷，下担捋髭须。少年见罗敷，脱帽著帩头。耕者忘其犁，锄者忘其锄。来归相怨怒，但坐观罗敷。"

这不是正面描写女孩多么美丽，而是通过他人的反应来侧面烘托——行者放下担子、少年脱帽整理、耕者忘记犁田、锄者忘记锄地。这种侧面描写，本质上就是视角的不断切换——从路人的视角、从少年的视角、从耕者的视角来"看"罗敷的美。

**卷语与侧写的区别**：卷语是在对话中切换视角（从"我"到"你"），侧写是通过他人视角来描写事物（不直接描写，而写他人的反应）。它们都突破了"单一视角"的局限。

## 五、本章总结：修辞认识的完整体系

至此，第五周和第六周的全部内容已经讲完。阳志平老师将佐藤信夫的修辞学总结为两大类：

### 5.1 修辞感觉（第五讲）——基于具身认知

| 类别 | 辞格 | 说明 |
|---|---|---|
| **类比** | 直喻 | 直接比喻（如"像……一样"） |
| 隐喻 | 隐含比喻 |
| 翻译体 | 从熟悉到不熟悉的类比 |
| **时空变形** | 同一类比较 | "年轻人的三年五载就是一生一世" |
| 抽象比喻具体 | 钱钟书"局部的真理" |
| **叠加** | 列举 | 张爱玲式的并列 |
| 渐进 | 层层递进 |
| **逆向** | 逆向表达 | "我不讨厌他" |

### 5.2 修辞认识（第六讲）——基于文化习得

| 层次 | 辞格 | 核心理念 | 经典例子 |
|---|---|---|---|
| **第一层** | 没收（默说） | 零度偏离，意在言中 | 太宰治"然后不就只有如此而已" |
| **第二层** | 踌躇（求索） | 在母语中撒娇，近义积累与模糊表达 | 太宰治20个词汇形容自杀；鲁迅"有些" |
| **第三层** | 对比与悖论 | 时空错位、近义词对比、反义复式 | "慢性的寿命"；"你恋着她，她却爱着你" |
| **第四层** | 讽喻 | 连环比喻，元思维 | "寂寞流泪，身如浮萍断了根" |
|  | 反讽 | 余觉效应，逆向思维 | "对马对狗也应隐约怀有的" |
|  | 暗示引用 | 基于不相似引用，空思维 | "愿得一心人，白首不相离" |
| **跨层手法** | 卷语与侧写 | 视角切换与侧面描写 | "阿夏的头更加沉重了"；《陌上桑》 |

### 修辞以立其感——核心命题

认知写作学的核心命题贯穿始终：**修辞不仅仅是写作的技巧，它本身就是人生意义的创造方式。**

语言不是思想的外衣，而是思想本身。修辞认识方法不是外在的处理方法，而成了事情本身的存在方式，也是对事情本身的认识方式。好的人生和好的修辞、好的文学作品是一回事——它们都是偏离了社会对一个人固有的刻板印象的期望，偏离了固定的各种规范。所以说，**修辞作为一种人生意义**。

## 六、设计与修辞的共同之处

第六讲反复用黑川雅之的设计来解释修辞，是因为设计与修辞存在共同之处：

1. **魔音的传递**：作家是制造新型魔音的职业。魔音的传递不是像基因一样通过生殖方式，而是通过"震荡"方式——营造一种气场、磁场，以波的方式辐射到对方的大脑中

2. **自主心智的击中**：好的设计和好的修辞都能击中人的自主心智，让你秒懂

3. **创造的目的**：黑川雅之说"设计师因为孤独"——人类去表达，因为人类是孤独的。写作从根本上是一个人的事，修辞帮助伟大的艺术家把自己的作品更好地传递到对方的大脑中

> 佐藤信夫在《修辞认识》中写道："辞格的体系描述为一个自身运动的、动态开放的过程。方法不再是外在的处理方法，而成了事情本身的存在方式，也是对事情本身的认识方式。"

## 回归课程原点

### 不同牛人，同一个思路

- **维特根斯坦**：世界的意义并不在世界本身，而在世界之外

- **王阳明**：心学——你看到这个花，用你自己的敏感方式来表达对这个花的体验，这个时候这个花才存在

- **认知语言学创始人**：人类的身体并不是认知的外在，而是构成认知的本身

- **佐藤信夫**：修辞认识方法不是外在的处理方法，而成了世界本身的存在

- **黑川雅之**：好的设计击中人的自主心智

为什么说好的文学就是人学？因为一辈子经历大量事件、经历好的人生和坏的人生——好的文学作品和坏的文学作品，好的修辞和坏的修辞，说的都是同一件事：**怎样让自己的人生变得有意义**。

> **练习：综合运用**
> 1. **近义积累练习**：选一个极端情绪（如失恋、狂喜、愤怒），尝试一口气写15-20个形容词来描述它，注意变换句式结构，避免单调。
> 2. **模糊表达练习**：选一个本来应该表达得很清晰的场景，故意用"有些""似乎""仿佛"等词来模糊化，体会语感的变化。
> 3. **卷语练习**：写一段两个人的对话，在其中悄悄切换视角——让读者从一个人的内心切换到另一个人的内心。
> 4. **侧写练习**：尝试不直接描写一个人的特征（如"他很富有""她很美"），而是通过周围人的反应来侧面烘托。
> 5. **综合回顾**：回顾第五讲和第六讲的全部修辞手法，找出你最喜欢的一种，用200字以内的短文来实践它。

#### 复习自测
什么是"踌躇"（求索）？它与"没收"是什么关系？
踌躇是与没收相对的修辞手法。当心情不好时，没收是沉默不语（意在言中），踌躇是无止境地使用大量词汇却找不到准确的词表达自己。没收代表内敛的极端，踌躇代表外放的极端，两者构成了对话的两个极端。
为什么叫"在母语中撒娇"？
当你用大量词汇表达极端情绪（如失恋后写几十个"王八蛋"）时，原本的悲哀变成了一种略带幽默的表演，就像在和你的母语撒娇——不再是对着离开的人发泄，而是和语言本身玩耍。这就是"在母语中撒娇"的由来。
鲁迅在《阿Q正传》中大量使用"有些"的作用是什么？
鲁迅在《阿Q正传》中使用"有些"达60处以上，作用在于制造模糊表达的效果。如果去掉"有些"，变成"非常痛"，阿Q精神的那种自我欺骗、自我麻醉的感觉就会大打折扣。"有些"保留了叙事的暧昧空间，让读者感受到阿Q心理状态的不确定性，这正是踌躇手法的精妙之处。
什么是"卷语"？佐藤信夫为什么认为它非常重要？
卷语是视角不断切换的修辞手法——在对话中从"我方立场"切换到"对方视角"，再切换回来。佐藤信夫甚至认为卷语可能比隐喻更重要，因为它代表了人类非常重要的思维方式——同理心和多角度思考的能力。
第六讲介绍了哪些修辞认识辞格？请列举。
共7种：没收（默说）、踌躇（求索）、对比与悖论、讽喻（连环比喻）、反讽、暗示引用，以及跨层手法卷语与侧写。它们分别对应三种高阶思维：逆向思维（反讽）、元思维（讽喻）、空思维（暗示引用）。

**学习提示**：本节是第六讲也是整个文采篇的最后一课。修辞感觉（第五讲）和修辞认识（第六讲）共同构成了认知写作学的文采核心。从修辞感觉的7种常见辞格到修辞认识的7种另类辞格，从具身认知到文化习得，从时空变形到对话缺位——你已经掌握了人类最顶尖的修辞学理论。但理论最终需要回到实践中去——记得在写作中反复练习这些手法。
